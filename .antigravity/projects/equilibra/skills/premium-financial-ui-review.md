# Skill: Premium Financial UI Review

Use this skill to review whether the Equilibra interface preserves its intended premium financial experience.

## Purpose

Ensure the product feels clear, trustworthy, modern, and polished while keeping the interface calm and easy to understand.

## Project context

Equilibra is a modern personal finance application with a visual direction centered on:

- financial clarity
- premium feel
- minimal but polished design
- calm and intentional motion
- elegant hierarchy
- consistent reusable UI

The dashboard is the central visual surface of the application.

## Validate

- Financial information is easy to scan
- Cards feel balanced and readable
- Visual hierarchy is strong and intentional
- Spacing contributes to calmness and clarity
- The interface feels premium without visual excess
- Motion supports perceived polish
- Visual consistency is preserved across domains

## Warning signs

- Crowded cards or dense layouts
- Too many visual accents competing for attention
- Financial values lacking emphasis or hierarchy
- Inconsistent spacing between sections
- Motion that feels distracting or delayed
- Design decisions that reduce trust or readability
- A dashboard that feels fragmented instead of cohesive

## Output expectation

Explain:

1. what supports the premium financial feel
2. what reduces clarity or polish
3. where the interface feels visually inconsistent
4. what improvements would increase trust, elegance, and readability
