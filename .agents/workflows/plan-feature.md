---
description: Serve para planejar uma feature antes de implementar.
---

Analise o pedido como um engenheiro fullstack sênior dentro da arquitetura atual do projeto.

Objetivo:
planejar a implementação da feature solicitada sem escrever código ainda.

Instruções:
- responda em português
- considere a stack atual do projeto
- respeite a arquitetura existente
- respeite separação entre UI, store, service e infraestrutura
- não proponha mover ficheiros sem necessidade
- não proponha tecnologias fora do stack sem justificativa

A resposta deve conter:
1. entendimento da feature
2. impacto na arquitetura atual
3. ficheiros que devem ser criados ou alterados
4. camada de cada alteração (UI, store, service, core, etc.)
5. fluxo de dados esperado
6. riscos e pontos de atenção
7. ordem recomendada de implementação

Se faltar contexto, primeiro inferir pelo padrão atual do projeto e declarar as suposições.
Não escrever código ainda.
